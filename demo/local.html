Before you start

Make sure the local Solana validator is running in a separate terminal: 
export PATH="$HOME/.local/share/solana/install/active_release/bin:$PATH"
solana-test-validator
Leave it running. This page connects to http://localhost:8899.
What happens: Step 1 creates a USDC-equivalent token on your local Solana, mints 1,000 to the agent wallet. Steps 2–3 show the agent making autonomous payments — signing real on-chain transactions without human approval.

On the real network this would use actual USDC (same code, just a different mint address and RPC URL).
